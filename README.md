# GrowthFlow SaaS Dashboard

A modern SaaS dashboard / web app UI built with React, Vite, Tailwind CSS, Framer Motion, and Lucide React icons.

## Features

- Responsive SaaS dashboard layout
- Mobile sidebar navigation
- Landing/hero section
- Analytics stat cards
- Revenue overview chart UI
- Recent activity panel
- Login UI card
- Profile settings UI
- Pricing section
- Clean component structure

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Deploy

You can deploy this project on Vercel.

1. Push this project to GitHub.
2. Go to Vercel.
3. Import the GitHub repository.
4. Click Deploy.
